Sobre Mim

Sou estudante de Engenharia de Software no 6º período. Estou constantemente em busca de aprimorar minhas habilidades como desenvolvedor, com foco na criação de interfaces funcionais e intuitivas para o usuário.

Atualmente, estou explorando o desenvolvimento mobile com Swift e SwiftUI, buscando criar interfaces modernas e fluídas. Além disso, trabalho com Python, MySQL, HTML, CSS e JavaScript, aplicando meus conhecimentos em tecnologias web para desenvolver soluções completas.

Acredito que um bom design deve facilitar a navegação e aprimorar a experiência do usuário. Possuo conhecimento em UX/UI e busco sempre aplicar esses princípios em meus projetos, garantindo interfaces eficientes e intuitivas. Além disso, sou capaz de integrar sistemas com APIs externas e automatizar processos para melhorar a usabilidade e otimizar o desempenho.

Neste portfólio, compartilho alguns dos meus projetos e aprendizados. Se você tem uma ideia ou projeto, sinta-se à vontade para entrar em contato!

Sistema de Pesquisa de Subpastas

Desenvolvi uma aplicação web com Flask e MySQL para realizar buscas rápidas por subpastas em um servidor corporativo contendo milhões de arquivos organizados em mais de 2 milhões de registros. A interface web possui uma área central rolável e responsiva, permitindo buscas por nomes completos ou parciais com alta eficiência.

Além da interface, implementei uma automação em Python responsável por varrer o sistema de arquivos do servidor, identificar todas as subpastas (inclusive em estruturas profundas) e registrar suas informações no banco de dados de forma estruturada. Esse processo é executado periodicamente e garante que a base de dados esteja sempre atualizada, mesmo com alterações constantes nas pastas da rede.

O projeto foi realizado de forma individual e é utilizado diariamente no ambiente de produção da empresa, facilitando o acesso e a organização de documentos e melhorando a produtividade da equipe.

MindFLow - Monitore Seu Humor

Para um projeto da faculdade, tive a ideia de criar um app (protótipo) de monitoramento de humor.

O aplicativo permite que o usuário relate como está se sentindo no momento e o motivo desse humor. Em seguida, é possível visualizar o histórico de humores durante a semana ou o mês, além de identificar a principal causa do sentimento. Também é possível compartilhar o histórico de humor com psicólogos, que poderão agendar consultas tanto online quanto presenciais. O protótipo foi criado no Figma e inclui telas para dispositivos móveis, além de uma versão web para psicólogos atenderem e analisarem seus pacientes.

Spotify Clone em SwiftUI

Desenvolvi uma interface inspirada no aplicativo oficial do Spotify utilizando Swift e SwiftUI. O projeto simula a tela de exploração de álbuns e playlists, com layout baseado em grid, cards reutilizáveis e uma navegação fluida seguindo os princípios modernos do design declarativo da Apple.

O foco do projeto foi aprimorar habilidades com SwiftUI, LazyVGrid, gerenciamento de estado e construção de UI responsiva, mantendo uma estética fiel ao Spotify com ênfase em espaçamentos, hierarquia visual e boa experiência do usuário.

Este projeto faz parte do meu repositório de estudos com SwiftUI, onde replico interfaces de apps reais para consolidar técnicas de animação, componetização e arquitetura visual limpa.
